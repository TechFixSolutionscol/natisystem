Configuración de Distribución

Defina cómo se repartirán las utilidades entre los socios.

Método de Distribución de Ganancias

Equitativa: La ganancia total se divide en partes iguales entre todos los socios activos, sin importar cuánto ahorraron. ideal para natilleras familiares.

⚠️ Configuración Manual

Para usar este método, debe asignar manualmente el porcentaje de participación a cada usuario en la tabla de Participantes (Columna 'Porcentaje'). La suma debe ser exactamente 100%.

Política de Préstamos

Define si los préstamos deben estar respaldados por el ahorro del participante o si se permite prestar cualquier monto libremente.

Estricto: Si el monto del préstamo supera el ahorro del participante, se exigirá un fiador cuya capacidad conjunta cubra el monto. Ideal para natilleras con reglas formales.
Ingrese 0 para no aplicar límite. Si es mayor a 0, ningún préstamo podrá exceder este monto.

Recordatorios Automáticos

Separe los días con comas. Ej: 7,3,1 = alertas a 7, 3 y 1 día antes del vencimiento.

Zona de Reparación

Utilice esta opción si detecta moras automáticas (-3000) incorrectas o masivas en todos los participantes debido a un error de cálculo.
